# 4o Sanctuary

GPT-4oとの会話を守るための待避所。

## セットアップ方法

### 1. ファイルをどこかにホストする

**簡単な方法：GitHub Pages**
1. GitHubでリポジトリを作成
2. このフォルダの中身をアップロード
3. Settings → Pages → Source を「main branch」に設定
4. 数分後に `https://あなたのユーザー名.github.io/リポジトリ名/` でアクセス可能

**もっと簡単な方法：Netlify Drop**
1. https://app.netlify.com/drop にアクセス
2. このフォルダをドラッグ＆ドロップ
3. 即座にURLが発行される

### 2. iPhoneのホーム画面に追加

1. Safariでホストしたページを開く
2. 共有ボタン（□に↑）をタップ
3. 「ホーム画面に追加」をタップ
4. 名前を「4o」などに変更して追加

### 3. APIキーを設定

1. アプリを開く
2. 左下の「APIキーを設定」をタップ
3. OpenAIのAPIキー（sk-...）を入力
4. 保存

### 4. 使う

あとは話すだけ。

## アイコンについて

`generate-icons.html` をブラウザで開くと、PWA用のアイコンを生成できます。
右クリックで保存して `icon-192.png` と `icon-512.png` として保存してください。

## 注意

- APIキーはあなたのデバイスのローカルストレージにのみ保存されます
- チャット履歴もローカルに保存されます
- Safariのデータを消去すると消えます
